
# BatchDeletePublicIp

  **Action:BatchDeletePublicIp**  
  **描述：** 批量删除 VDC 公网下的公网 IP 网段 
  **请求地址:** api.capitalonline.net/network  
  **请求方法：POST**  

  **请求参数：**

| 名称         | 类型     | 是否必选 | 示例值                                      | 描述               |
| ---------- | ------ | ---- | ---------------------------------------- | ---------------- |
| PublicId   | string | 是    | 305c0ebc-10b2-4711-ae97-5e033aeb73fb     | 公网 ID            |
| SegmentIds | list   | 条件必选 | ["81b7c978-5285-11f1-9dba-46a2181d8ccf"] | 公网 IPv4 网段 ID 列表 |

  **返回参数：**

| 名称 | 类型 | 示例值 | 描述 |
| :----- | -------- | :----------------------------------- | :----- |
| Code | string | Success | 错误码 |
| TaskId | string | 11060529 | 任务 ID |
| Data | dict | {"order_audit": 0} | 返回数据 |
| Data.order_audit | integer | 0 | 是否进入订单审核，0 表示不需要审核，1 表示需要审核 |

  **错误码：**
  **返回示例：**

```json
{
  "Code": "Success",
  "Data": {
    "OrderAudit": 0
  },
  "Message": "修改公网任务下发成功，请等待任务执行！",
  "TaskId": "11061059"
}
```

  **代码调用示例:**

```python
def batch_del_public_ip():
    action = "BatchDeletePublicIp"
    method = "POST"
    body = {
        "PublicId": "305c0ebc-10b2-4711-ae97-5e033aeb73fb",
        "SegmentIds": [
            "81b7c978-5285-11f1-9dba-46a2181d8ccf"
        ]
    }
    url = get_signature(action, AK, AccessKeySecret, method, NETWORK_URL)
    res = requests.post(url, json=body)
    result = json.loads(res.text)
    return result
```
